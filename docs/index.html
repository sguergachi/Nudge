How it works

1

Track

Nudge silently watches your foreground app and idle time — no screenshots, no cloud, no accounts.

2

Learn

Each time you answer "productive?" the on-device model gets smarter. After ~50 answers it starts predicting drift in real time.

3

Nudge

When the AI detects you're drifting it pops a quick check-in. When you're focused it stays silent. No noise, no fixed timers.

Quick start

Windows

  1. Download nudge-win-x64.zip
  2. Extract and run nudge-tray.exe
  3. A tray icon appears — you're tracking

For ML: pip install scikit-learn joblib pandas numpy, then enable ML in Settings.

Linux

  1. Download nudge-linux-x64.tar.gz
  2. tar -xzf nudge-linux-x64.tar.gz
  3. Run ./install-linux-app.sh, then launch nudge-tray

Supports Wayland (KDE, GNOME, Sway) and X11.

Privacy first

Everything stays local. Nudge writes to ~/.nudge/ on your machine. There is no server, no telemetry, no account. Your productivity data is yours.
