02

AI Brain

Your personal AI Brain — trained exclusively on your YES/NO responses.

Runs inference at sub-100ms — asks only when confident you're drifting.

Activates after 100 samples; retrains automatically at every 50 new responses.

Adaptive architecture: lightweight (<200 samples), full feature set (200+).

Sub-100ms inference · TCP 127.0.0.1:45002 · retrains automatically · no cloud
03

Adaptive Interrupt

No fixed schedule — fires only when drift confidence hits ≥98%.

Stays silent when productive; falls back to variable-interval below threshold.

Ctrl+Shift+Y / Ctrl+Shift+N respond without leaving the keyboard.

30s auto-dismiss — every answer feeds back to sharpen the model.

≥98% confidence · 30s auto-dismiss · UDP 127.0.0.1:45001

Includes a live analytics dashboard accessible from the system tray — track daily, weekly, and monthly focus trends.
