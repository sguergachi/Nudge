Desktop attention state estimation.

21-feature snapshots evaluated at 60s intervals against a local scikit-learn binary classifier trained on your YES/NO response history. Inference via TCP 127.0.0.1:45002. Interrupt at ≥98% drift confidence, or fall back to variable-interval sampling.

Pipeline

Signal acquisition
Foreground application, idle duration, window title, and compositor metadata sampled at 1 Hz. Browser domain extracted from window title when a browser is focused. No screenshots, no keystroke logging, no network egress.
Source: KWin Script / X11 EWMH / Sway IPC / Win32 Quality: Trusted | Usable | Poor
Feature extraction
Each signal classified into one of six categories (Development, Creative, Office, Communication, Entertainment, Utility) at five confidence tiers — from OS ground truth to sensible defaults. A rolling 300 s window tracks switch counts, distinct apps, application share, and anchor-return events.
Output: 21-dimensional feature vector Window: 300 s
Inference
Vector evaluated against a scikit-learn classifier served over TCP 127.0.0.1:45002. Requires 100 training samples before the model activates; retrained by a background process at 50+ new samples via 300 s polling.
Threshold: ≥98% confidence Interval: 60 s
Conscious interrupt
A floating window with a single binary choice: YES or NO. Keyboard shortcuts (Ctrl+Shift+Y / Ctrl+Shift+N). 30 s auto-dismiss. Every response is appended to the training set and triggers the next retrain cycle.
Window: custom Avalonia overlay IPC: UDP 127.0.0.1:45001
Stack
  • Harvest daemon: C# / .NET 10
  • UI: Avalonia (tray + notification)
  • Classifier: scikit-learn (Python)
  • IPC: TCP 45002 / UDP 45001
Privacy
  • All data in ~/.nudge/
  • No accounts, no telemetry
  • No network calls from daemon
  • Open source (MIT)
Supported
  • Windows 10 / 11
  • Linux: KDE, GNOME, Sway, Cinnamon
  • Wayland + X11
  • x86_64 only
